Day 1 · 13/5

昆明 → 大理

凌晨抵达昆明 → 高铁前往大理 → 喜洲古镇 + 大理古城

喜洲古镇

中午（11:00–14:00）

  • 转角楼：白族建筑代表，经典打卡点。
  • 董金汉院：有“我喜洲”“欢喜之洲”字墙。
  • 三水与柏：特色的杂货铺，木质门头和小物件。
  • 喜棠：有“喜洲”大字打卡墙，人相对少一些。
  • 天工坊：《去有风的地方》取景地

下午（14:00–17:00）

下午重点在麦田区域拍照，从田野一路走到咖啡馆，整体节奏比较轻松。

麦田（田野风景）4–5 月是绿麦田 → 喜林苑黄墙（热门机位） → 有风号小火车（拍照体验） → 路边野餐·咖啡画廊（咖啡 / 孤独小屋）
  • 麦田：主要风景区。
  • 喜林苑黄墙：热门的拍照机位。
  • 有风号小火车： rmb35/person
  • 路边野餐·咖啡画廊：孤独小屋 + 咖啡 。离古镇有一段距离，要打车 ~15分钟

推荐路线

正义门（古镇入口） → 转角楼（白族建筑） → 董金汉院（字墙拍照） → 三水与柏（特色杂货铺） → 喜棠（打卡墙） → 麦田（田野风景） → 喜林苑黄墙（热门机位） → 有风号小火车（拍照体验） → 天工坊（取景地） → 路边野餐·咖啡画廊（咖啡 / 孤独小屋）

大理古城

晚上看灯，吃晚餐， chill。

傍晚 & 夜晚（18:00–22:00）

整体走一圈大概 3km 左右，不需要赶时间。

南门 → 南诏十二时辰 → 红龙井 → 人民路
  • 南门：夜景起点。
  • 南诏十二时辰：灯笼街区（19:30–22:00）。
  • 五华楼：古城地标，可以拍夜景。
  • 洋人街 / 人民路：吃东西和夜生活比较热闹。
  • 红龙井：水景酒吧街，可以坐着听歌、喝点东西。

Day 2 · 14/5

大理 → 沙溪

大理出发 → S湾拍照 → 前往沙溪古镇 → 住沙溪

S湾

环海公路 + 吉普车旅拍。

上午（09:00–12:00）

早上从大理出发到S湾， 有安排吉普车旅拍在这里完成。 骑脚车环岛

沙溪古镇

更安静的古镇，适合慢慢走 + chill chill chill。

下午（14:00–17:00）

逛古镇核心区， 可以随便走走拍照。

Day 3 · 15/5

沙溪 → 丽江

前往丽江 → 白沙古镇 → 附近景点拍照 → 晚上看丽水金沙表演（送的）

白沙古镇

更安静的古镇 + 雪山背景 + café 打卡。

中午（11:00–14:00）

可以慢慢走 + 拍照 + 找 café 坐。

附近拍照点

拍照地点+路线。

下午（14:00–17:00）

  • 见山小院：比较出名的 café。
  • 云堤岸咖啡庄园
  • 江大经幡

丽江古城 & 表演

晚上逛古城 + 看丽水金沙表演。

Day 4 · 16/5

丽江 → 香格里拉

从丽江-> 香格里拉 长江第一湾 → 虎跳峡 → 藏家土司宴 → 独克宗古城 → 住香格里拉

长江第一湾，虎跳峡

scenic route

白天（10:00–16:00）

从丽江出发前往香格里拉，沿途会先经过长江第一湾，再去虎跳峡。 路程也会比较多。

长江第一湾（观景点） → 虎跳峡（自然景观）
  • 长江第一湾
  • 虎跳峡

香格里拉体验

晚餐 + 古城夜景。

傍晚 & 夜晚（18:00–21:00）

到达香格里拉后，晚上会先体验藏家土司宴， 然后再去独克宗古城走走。

Day 5 · 17/5

香格里拉 → 丽江

转经筒 → 坛城 → 普达措森林公园 → 返回丽江

香格里拉市区

上午（09:00–11:30）

转经筒和坛城， 节奏不会太赶。

转经筒（地标） → 坛城（文化景点）
  • 转经筒：香格里拉地标，可以转一圈体验一下。
  • 坛城：藏族文化建筑，拍照也很好看。

普达措森林公园

湖 + 森林（今天的重点）

中午 & 下午（12:00–16:00）

主要是看湖和森林景色， 可以走轻松路线，不需要全部走完。

  • 湖景
  • 森林步道：可以走一小段，不用走太多。
  • 整体是放松型行程。

返回丽江

结束香格里拉行程，回到丽江。

Day 6 · 18/5

丽江 → 昆明

玉龙雪山 / 云杉坪 → 蓝月谷 → 返回丽江 → 动车回昆明

玉龙雪山

今天的重点景点（雪山 + 高海拔）

上午（08:00–12:00）

早上前往玉龙雪山，云杉坪小索道， 主要是看雪山景色 + 拍照，节奏不用太赶。 高海拔，如果不舒服就不要勉强上高点。不要跑跳太多。

蓝月谷

中午（12:00–14:00）

从雪山下来后前往蓝月谷，湖水颜色很好看， 适合走走 + 拍照。

  • 可以慢慢走，不需要走太远。

返回昆明

结束丽江行程，返回昆明。
